# Agua La Mina

Sistema web para la administración de pedidos de agua purificada.

## Objetivo

Permitir que cajeros, repartidores y administradores gestionen pedidos desde teléfonos móviles.

## Roles

### Administrador

* Consultar ventas
* Consultar clientes
* Consultar historial
* Consultar préstamos de garrafones

### Cajero

* Registrar clientes
* Registrar pedidos
* Consultar historial de clientes

### Repartidor

* Consultar pedidos pendientes
* Abrir ubicación en Google Maps
* Marcar pedidos entregados
